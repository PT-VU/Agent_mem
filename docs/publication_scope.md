# Publication Scope

## Included

- Agent-mem v2 source used by the core F3 experiment.
- Multi-agent source extending v2 with T1-A, T1-B, and T1-C.
- Shared experiment orchestration and evaluation helpers.
- Fixed instance lists, active configs, run manifests, attempt summaries, and
  official evaluation records for the reported runs.
- Concise English reports and the current LNCS thesis source.

## Excluded

- Virtual environments, caches, compiled files, runtime locks, PID files, tmux
  state, and Docker artifacts.
- Raw trajectories and large event logs.
- Runtime `.env` files and API credentials.
- External reference PDFs.
- Unrelated early exploratory branches and long internal notes.

The snapshot is intentionally small enough for public review while retaining
the code and evidence needed to inspect the reported thesis experiments.
